SOS

In this section, the user will be able to call out for assistance. The SOS button will contact the emergency services, such as the police, as well as their emergency contacts and will send these ones the live location of the user. In addition, there will be an optional feature the user will be able to decide if they want to allow or not: in case the SOS button is activated, the frontal camera and microphone would start recording everything so the victim could use these as evidence in case they would like to take the aggression to court.

Alert mode

"This would be a similar feature to SOS but, for this one, only the emergency contacts would be warned. The user could use this feature in case they feel they could be in a situation in danger, but they are not completely sure of it. They could activate the alert mode in situations like walking home and feeling someone approaching them for a while. This could be a dangerous situation but it could also be nothing so, if the user does not feel like contacting the police, they could warn their emergency contacts to let them know they could be in a situation of danger and would send them their current location.

QR Scanner

This function has been designed as a partnership with night clubs and festivals. From BfreeLabs we know that harassment can happen in the middle of a crowd, this is why, when going to these places, there are going to be QR codes that the user will scan. Once scanned, the SOS button would also contact the security staff from the clubs or festival, assuring a faster assistance.

Also, the app will partner up with other app companies as Cabify. The main purpose of this alliance would be to verify some of this lift’s app drivers to let know BFree users that they are going to get on a safe ride. This is because from BFree Labs we know sometimes the most dangerous part of a night out is going back home and we want to minimize the chances that you don’t make it home safely.

Emergency contacts

This functionallity will allow the user select which contacts they would like to notify in case of an emergency. These contacts would receive a notification as well as the live position of the user in case this one triggers the BFree alert.

Wrist band

From this section, the user will be able to link the phone app with either their BFreeBand or smartwatch for a quicker access to BFree assistance in case of an emergency.
